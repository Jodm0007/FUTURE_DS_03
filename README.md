# Bank Marketing Funnel Analysis Project

## Project Structure

This project analyzes marketing campaign data from the Bank Marketing dataset to understand the lead-to-customer conversion funnel, identify drop-off points, and evaluate channel performance.

### Directory Structure

```
FUTURE_DS_03/
├── data/
│   ├── raw/                     # Original datasets
│   │   └── bank-marketing.csv
│   └── processed/               # Cleaned and processed data
│       └── bank-marketing-cleaned.csv
│
├── notebooks/                   # Jupyter notebooks for analysis
│   ├── 01_data_cleaning.ipynb   # Data cleaning and preprocessing
│   ├── 02_funnel_analysis.ipynb # Funnel metrics and drop-off analysis
│   └── 03_key_insights.ipynb    # Channel performance and recommendations
│
├── reports/                     # Final reports and visualizations
│   ├── funnel_analysis_report.md
│   └── images/                  # Generated charts and funnel diagrams
│
├── tableau/                     # Tableau/Power BI dashboards
│   └── funnel_dashboard.twbx
│
├── src/                         # Reusable Python scripts
│   ├── data_cleaning.py
│   ├── funnel_metrics.py
│   ├── funnel_analysis_functions.py
│   ├── utils.py
│   └── visualization_utils.py
│
├── F_DS_03_env/                 # Virtual environment
├── README.md
└── requirements.txt
```

## Analysis Objectives

1. **Traffic-to-Lead Conversion**: Analyze the effectiveness of generating initial interest from campaigns.
2. **Lead-to-Customer Conversion**: Evaluate the success rate of closing prospects into subscribers.
3. **Funnel Drop-offs**: Identify the specific stages where potential customers disengage.
4. **Channel Performance**: Compare conversion efficiency across different communication styles, durations, and profiles.

## Getting Started

1. Create the virtual environment (if not already done): `python -m venv F_DS_03_env`
2. Activate the environment (`.\F_DS_03_env\Scripts\activate` on Windows).
3. Install dependencies: `pip install -r requirements.txt`
4. Make sure the dataset `bank-marketing.csv` is present in `data/raw/` directory.
5. Explore the Jupyter notebooks in the `notebooks/` folder sequentially.
